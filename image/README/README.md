# Aurora 演示图片目录

这个目录用于存放 README 中展示的软件截图。

建议命名：

- `dashboard-overview.png`
- `knowledge-base.png`
- `chat-workbench.png`
- `knowledge-graph.png`
- `logs-settings.png`

使用方式：

1. 把演示截图放到当前目录。
2. 打开根目录 `README.md`。
3. 将“界面预览”表格中的“待补充截图”替换为对应的 Markdown 图片链接。

示例：

```md
![](./image/README/dashboard-overview.png)
```
